DS Med Cabinet API Address

API Usage

Send a POST or PUT request to https://ds-med-cabinet.herokuapp.com/predict with a json object structured like so:

        {
        "id": "420", 
        "First Name": "John", 
        "Last Name": "Doe",
        "Desired Effects": "Uplifed, Happy, Relaxed, Energetic, Creative"
        }

The API response will PUT a JSON object with recommended Strain and Effects to the Med Cabinet Database where it can be retrieved by Med Cabinet members.
