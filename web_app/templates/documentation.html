DS Med Cabinet Recommendation API URL Address

https://ds-med-cabinet.herokuapp.com/predict

Recommendation API Usage

Send a [POST] or [PUT] request to https://ds-med-cabinet.herokuapp.com/predict with a json object structured like so:

        {
        "id": "420", 
        "Desired Effects": "Uplifed, Happy, Relaxed, Energetic, Creative"
        }

The API response will PUT a JSON object with recommended Strain and Effects to the Med Cabinet Database where it can be stored and retrieved by Med Cabinet members via Front End Web services.

There is also a local strain recommendation service page with GUI located in the navigation bar or also available at this link:

https://ds-med-cabinet.herokuapp.com/recommendations


Request API Usage

Send a [GET] request to https://ds-med-cabinet.herokuapp.com/request to request all of the Med Cabinet API Data.


Data API Usage

Also try to Send a [GET] request to https://ds-med-cabinet.herokuapp.com/data to request all of the Med Cabinet API Data. This page has a GUI, so you may prefer the above mentioned API request


Strain in URL API Usage

Send a [GET] request to https://ds-med-cabinet.herokuapp.com/ and replace "strain" with a strain name to request data for that specific strain. Strain names can be found in the previous two API's that are mentioned or also in the Strain Table and Strain Menu


Strain Table and Strain Menu

Although these two pages are not API's and this is an API first application, it is worth noting that these two pages were added for non technical users to explore the data by using the find functions on their respective operating systems. The GUI here is more user friendly and the links can be found in the top Navigation Bar.


Strain in URL Menu

This page is also found in the Navbar and is not an API, yet it includes functionality to search for a strain using the URL.
